# webpack-starter
Webpack 5 starter

# How to install?

- Open the terminal and run following commands.

  ```
  git clone https://github.com/rdshinde/webpack-starter.git
  ```
  
  ```
  npm install
  ```
  
  ```
  npm run build
  ```
  
  ```
  npm start
  ```
  
  This will start the application on port no. 3000
